## Theme & Chart Type Coverage

The sample exports demonstrate the application's capability to export content in:

### Themes
- **Light Mode**: Clean, professional appearance with light backgrounds
- **Dark Mode**: Modern dark theme with high contrast for better visibility

### Chart Types  
- **Area Charts**: Filled area visualization showing trends over time
- **Bar Charts**: Vertical bar visualization for easy comparison
- **Line Charts**: Connected points showing detailed trend progression

## Usage

These files serve as:
- **Examples** of expected output quality and format across different configurations

- **Documentation** showcasing the application's visual and export capabilities

- **User demonstrations** for showcasing export flexibility

## Data Source

All sample exports are generated from the test dataset containing 60 rows of financial data spanning multiple departments (Sales, Marketing, Support), regions (North America, Europe, Asia Pacific), and product categories, covering the period from 2024-01 to 2025-08.
